# URL Shortner Take Home Challenge

<p>For this task, I was asked to use Docker-Compose, React and Node.js to create the UI and backend</p>
<br>
<p>To run:</p>
<ul>
<li> clone repo</li>
<li> Install dependecies: </li>
<ul> 
<li> ejs </li>
<li> express </li>
<li> http-errors </li>
<li> mongoose </li>
<li> shortid </li>
</ul>
<li> docker-compose build && docker-compose up </li>
</ul>
